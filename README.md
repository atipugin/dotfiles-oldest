# dotfiles

My cozy collection of system dotfiles and useful stuff.

## Usage

1. First of all, install [Homebrew](http://brew.sh/).

2. Clone repository:

```ssh
git clone https://github.com/atipugin/dotfiles.git
```

3. Install brews:

```ssh
cd dotfiles && brew bundle
```

4. Run rcup!

```ssh
rcup -d `pwd` -x README.md
```
